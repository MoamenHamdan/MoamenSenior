## AI Features

The M-Suite application has been enhanced with artificial intelligence capabilities:

### Chatbot Assistant

The M-Suite Assistant is an AI-powered chatbot that helps users navigate the system and find information quickly. Features include:

- Intent recognition to understand user queries
- Guided assistance for creating transactions
- Information about items and inventory
- Finding related products for cross-selling
- Simple natural language interface

### Item Correlation Analysis

This feature uses machine learning to identify items that are frequently purchased together. Benefits include:

- Discover strong product relationships to improve cross-selling
- Optimize product placement in physical stores
- Create effective bundles and promotions
- Enhance customer experience through relevant suggestions

### AI Dashboard

The AI dashboard provides a central location to access all AI features and insights. It displays:

- Item correlation analysis
- Transaction pattern insights
- Customer segmentation
- Anomaly detection

### Technical Implementation

The AI features are implemented using:

- ML.NET for machine learning capabilities
- Matrix Factorization for item correlation analysis
- Intent recognition for chatbot functionality
- RESTful APIs for frontend integration
